Voice Recognition App with Benevity

Khoi Ngo and Branden Shin

In 2018, the Gartner Research group predicted that roughly 30% of all interactions with technology will be performed via conversations with smart machines [1]. Thus, we have seen a plethora of tech corporations such as Amazon, Google, Apple, Microsoft and Samsung (to name a few) adopt and invest in their voice recognition technology. As this technology continues to grow, the market for it grows as well [2].

This project is sponsored by the company, Benevity. Benevity provides a corporate social responsibility platform that promotes and facilitates charitable giving and volunteering efforts for companies and their employees. Benevity has given our group the opportunity to incorporate voice recognition technology with their API, in an effort to improve the user experience.

As a general overview of this project, we have developed an Android app that uses Google Assistant voice commands to use the Benevity API and search for causes. A user can search for causes using either a keyword or a location (or both), and the results are displayed in the app. Once the results are displayed, the user is then able to navigate the list of results using built-in voice commands. Figure 1 below shows a typical use case of this app, where a user launches the Google Assistant and uses a voice command to search for causes. They then make a subsequent voice command to fetch more details about a cause.

Figure 2 shows the views of our app. We can see the main page has a basic search query where user can search by keyword and/or location. The next view will display the results if found. The user can select a cause they wish to learn more about. After clicking a cause, the information of that cause will be displayed to the user. 

To summarize, we developed an Android app that can retrieve data using the Benevity API. This app can be launched with the Google Assistant to vocally search for causes in a more convenient manner. Additionally, the user can navigate through the results returned by the API using voice commands, giving the app an almost chat-bot like feel. The app was converted into an Android Library to promote code reuse in the future.

With the rise in voice recognition technology, we believe it is beneficial for Benevity to begin investing resources into it to provide an added convenience to their customers. Voice recognition technology is growing continuously, and the leading companies (such as Apple’s Siri and Google Assistant) continue to improve their voice recognition products. Our project provides evidence that it is possible for Benevity to integrate voice recognition into their products to improve the user experience.

Summary of Features

  • Search by keywords and/or location through Google Assistant
  • Search by keywords and/or location through voice commands in app
  • Search by keywords and/or location through typing in app
  • Use voice commands to move through app
  • App outputs information to user
  • Accessible app for visually impaired
